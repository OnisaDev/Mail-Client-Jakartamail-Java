# Mail-Client-Jakartamail-Java
Cliente de Correo SMTP con Jakarta Mail y Java 21

Descripción: 
Módulo de comunicación para un software de gestión farmacéutica que permite el envío automático de alertas de stock y pedidos.


Características:
Soporte para protocolos de seguridad SSL (puerto 465) y TLS (puerto 587).

Gestión de múltiples destinatarios.

Implementación modular y flexible.



Tecnologías: 
Java 21, Maven, Jakarta Mail API
